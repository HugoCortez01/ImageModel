# Test Assistent Programming

## Descrição

Este projeto é uma coleção de exemplos de código Python desenvolvidos para fins educacionais e de prática em programação. Ele inclui implementações de algoritmos básicos, códigos com erros intencionais para depuração, e exemplos de refatoração para melhorar a qualidade do código. Cada script é acompanhado de uma explicação detalhada em arquivos Markdown.

O objetivo é demonstrar conceitos fundamentais de programação, como entrada de dados, validação, cálculos matemáticos, tratamento de erros e boas práticas de codificação em Python.

## Estrutura do Projeto

- `num.primo.py`: Script para verificar se um número é primo.
- `debug.py`: Calculadora de compras com erros de sintaxe e lógica para prática de depuração.
- `refatoracao.py`: Exemplo de refatoração de código para calcular estatísticas de uma lista de números.
- `explicacao_num_primo.md`: Explicação detalhada do algoritmo de verificação de números primos.
- `explicacao_debug.md`: Análise linha por linha do código `debug.py`, destacando erros e conceitos.
- `explicacao_refatoracao.md`: Descrição das melhorias aplicadas na refatoração do código.

## Pré-requisitos

- **Python 3.x**: Certifique-se de ter o Python instalado. Você pode baixá-lo em [python.org](https://www.python.org/).
- Nenhum pacote adicional é necessário, pois os scripts usam apenas bibliotecas padrão do Python.

## Como Usar

1. **Clone ou baixe o repositório**: Copie os arquivos para o seu ambiente local.

2. **Execute os scripts**:
   - Abra um terminal na pasta do projeto.
   - Execute os comandos abaixo para cada script:

     - Para `num.primo.py`:
       ```
       python num.primo.py
       ```
       - O programa solicitará um número inteiro e informará se ele é primo.

     - Para `debug.py`:
       ```
       python debug.py
       ```
       - **Atenção**: Este script contém erros intencionais. Ele pode falhar ao executar. Use-o para prática de depuração (consulte `explicacao_debug.md` para detalhes dos erros).

     - Para `refatoracao.py`:
       ```
       python refatoracao.py
       ```
       - O programa calcula e exibe estatísticas (total, média, maior e menor) para uma lista de números de exemplo.

3. **Leia as explicações**: Para entender melhor cada script, consulte os arquivos `.md` correspondentes.

## Exemplos de Execução

```

O número 17 é primo.
```

### debug.py
- Devido aos erros, a execução pode resultar em mensagens de erro como `SyntaxError` ou `TypeError`. Use isso como exercício para corrigir o código.

## Conceitos Demonstrados

- **Verificação de números primos**: Algoritmo eficiente com loop otimizado.
- **Entrada e validação de dados**: Tratamento de entradas inválidas.
- **Cálculos matemáticos**: Somas, médias, impostos e descontos.
- **Depuração**: Identificação e correção de erros comuns em Python.
- **Refatoração**: Melhoria de código para legibilidade, usando estruturas como `NamedTuple`, funções separadas e validações.
- **Boas práticas**: Uso de docstrings, tipagem, separação de responsabilidades e nomes descritivos.
